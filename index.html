SMART ATTENDANCE SYSTEM

Submitted by: JANAKI.R (AM.SC.U4CSE24121)

Abstract

This project involves designing a Fingerprint Sensor-Based Biometric Attendance System using Arduino. It integrates a fingerprint sensor, LCD display, and RTC module. The system records and manages attendance data using biometric identification to ensure secure and accurate tracking.

Introduction

Background

Biometric systems are increasingly used for secure identification. In schools and workplaces, traditional attendance systems often suffer from inaccuracies and fraud. This project introduces fingerprint-based attendance to overcome these issues.

Problem Statement

Traditional methods like manual roll calls and sign-in sheets are inefficient and prone to errors. This project aims to eliminate these shortcomings with a reliable biometric system.

Objectives

  • To design and implement a reliable biometric attendance system using Arduino.
  • To ensure accurate and tamper-proof attendance records.
  • To develop a user-friendly interface for administrators and users.
  • To evaluate the system's real-world performance.

Significance

The system improves accuracy, saves time, reduces workload, and increases trust in attendance recordsâ€”making it beneficial for educational and professional environments.

Hardware Setup

  • Arduino Microcontroller: Central control unit.
  • Fingerprint Sensor: Captures fingerprint data.
  • LCD Display: Provides real-time feedback.
  • Database: Stores user data and attendance logs.

Software Development

  • Code Implementation: Developed using Arduino IDE.
  • Algorithm Design: Matches fingerprints and logs attendance.
  • Testing: Ensures accuracy and reliability.

Implementation

The system architecture includes hardware components (Arduino, fingerprint sensor, LCD, database) and is built through iterative design and testing. It emphasizes ease of use, precision, and durability.
